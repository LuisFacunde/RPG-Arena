# RPG Arena

Projeto final desenvolvido para o curso de férias de **TypeScript** da **CESAR School**.

## Objetivo
Este projeto consiste em um sistema de batalha RPG simples, criado com o propósito de aplicar conceitos fundamentais de **Orientação a Objetos (POO)** e **TypeScript**, incluindo:
* Classes e Herança
* Interfaces e Polimorfismo
* Encapsulamento (public/private/protected)
* Tratamento de Erros Personalizados

## Funcionalidades
* **Personagens:** Classes especializadas (`Guerreiro`, `Mago`, `Arqueiro`) com atributos e habilidades únicas.
* **Sistema de Batalha:** Lógica de turnos dinâmica onde personagens lutam até que reste apenas um vencedor.
* **Itens:** Implementação de inventário com poções de vida e mana.
* **Inteligência Artificial Simples:** Script de batalha onde os personagens tomam decisões baseadas em seus status de vida e mana.

## Pré-requisitos para utilizar o projeto
* **Node.js** versão 14.0.0 ou superior
* **npm** versão 6.0.0 ou superior
* **TypeScript** - instalado como dependência

## Como Executar

1.  Clone o repositório.
    ```bash
    git clone https://github.com/LuisFacunde/RPG-Arena.git
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Execute o projeto:
    ```bash
    npm start
    ```
